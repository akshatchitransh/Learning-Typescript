ABSTRACT CLASSES
1- ek class hi hai 
2- used to extend class
3- iske andar function directly bhi likh sakte hian 
4- iske andar agar abstract karke function likhte ahin to uski body ni likhte hain , vo har subclass determine karta hai jese meow , bhow bhow.



abstract class MyAbstractClass {
    abstract sayHi: () => void;
}

class MyClass extends MyAbstractClass {
    greet(): string {
        return "Hello from subclass";
    }

    sayHi = () => {
        console.log("Hi from subclass");
    };
}
